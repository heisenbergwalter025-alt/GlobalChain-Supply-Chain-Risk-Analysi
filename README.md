# 🌐 GlobalChain Ltd: Supply Chain Risk & Resilience Analysis

## 📖 Project Overview
This project focuses on identifying vulnerabilities and simulating disruption scenarios for **GlobalChain Ltd**, a multinational electronics manufacturer. The goal was to build a data-driven resilient strategy using Python and Power BI.

## 🛠 Tech Stack
- **Python:** NetworkX (Network Analysis), Scikit-Learn (K-Means Clustering), Statsmodels (ARIMA Forecasting), SciPy (Monte Carlo Simulation).
- **Power BI:** 4-page interactive dashboard for risk communication.

## 🚀 Key Insights
- **Network Analysis:** Identified Factory F3 as a critical bottleneck (highest Betweenness Centrality).
- **Supplier Profiling:** Clustered 25 suppliers into 4 risk tiers using Machine Learning.
- **Demand Forecasting:** Modeled 12-month demand with uncertainty bands (Monte Carlo).
- **Risk Simulation:** Calculated Value at Risk (VaR) and ROI for resilience investments.
